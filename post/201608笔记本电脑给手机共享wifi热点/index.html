笔记本电脑给手机共享wifi热点

27 Feb 2014, 13:56
读书学习
wifi

适用范围：没有wifi，仅有连接网线上网的手提电脑。使用以下步骤配置后，手机、pad等可以使用笔记本共享的wifi上网。

  1. 打开笔记本的无线网络开关

  2. 以管理员身份打开“命令提示符”

  3. 依次输入下面2行命令

    netsh wlan set hostednetwork mode=allow ssid=T410 key=1234567899
netsh wlan start hostednetwork
说明： ssid=T410 表示无线的名称为T410，key=1234567899表示无线密码为1234567899

  4. 打开网络连接能看到一个新增的无线网络连接

  5. 在本地连接上右键，然后点属性，将本地连接共享给刚才创建的 “无线网络连接 2”

  6. 完成了所有配置。每次电脑关机、开机后，重复步骤1、2，然后输入一行命令：netsh wlan start hostednetwork

（说明：可以将此命令保存成bat格式的文件，需要时双击执行或者设置成开机自动执行，见下图）
