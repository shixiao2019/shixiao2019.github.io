筷子的礼仪

21 Aug 2016, 14:56
读书学习
文化 / 筷子

筷子的礼仪 客人入座后，切不可急急地动筷。只有等主人先握筷，向大家比划一下，以筷相邀宾客说“诸位不用客气，大家请！”之后，客人方可动筷。若遇重大的宴席，须等上首的长辈先动筷，并象征性地夹上一筷后，其它人才能动筷。 餐毕，先毕的客人要用筷向全桌未毕者示意性地指 餐毕落筷，大家才能将筷儿从碗口取下，放于桌上，再离座话别。 无论节日宴还是喜宴，待客宴，“碗口筷”是犯大忌的。“碗口筷”，即把筷子平放在碗口上。这种“人不陪席筷陪席“的礼节，如果运用的时机不当，说不定会闹得主客失和。主人在宴请亲朋好友若一开席就放了“碗口筷”，那就等于下了逐客令。因为那是民间祭祀游魂野鬼时的仪式，用于待客，就是在奚落客人来讨饭吃了。如果客人在餐毕离坐后没有把筷子从碗口取下，放于桌上，则是表示还没有吃饱，对主人是大不敬的。但是，最忌“碗口筷”的，还要数海上的渔民，若在船上吃饭，饭毕把筷子平放在碗口上，那就意味着“桅杆横倒船要翻” “长短筷”也是不行的。“长短筷”指的是所用的筷子一长一短。作为主人，在摆筷子的时候，千万注意，筷子不能一长一短。民间认为这是在诅咒客人夫妇或他的父母不能“双双同到老”，而是“一长一短”。 “杂色筷”也是不行的，既两根筷子的颜色，式样，用料不一样，杂色筷意味着家庭不和睦，经常闹别扭。主客双方都是大忌。 “截筷”和“泪筷”也为人所不屑。“截筷”有两种，一种是好客的主人把菜夹给客人时，客人用筷在空中把主人夹过来的菜接走；一种是俩人同时出筷，同伸向一个菜盘的同一块菜肴。“泪筷”即夹带汤的菜，一路滴滴答答地掉个不停。 “插筷”是最犯忌的，即在用饭前把一双筷子直插于饭中或者菜中。在江南地区，只有给死人上供的时候，才有这样的插法。 筷礼八忌 一忌迷筷：举筷不定，东戳西夹。 二忌翻筷：以筷翻挖，菜底拣食。 三忌刺筷：夹菜不住，改之为刺。 四忌涮筷：以筷搅汤，左右捞物。 五忌舐筷：放筷于口，吮吸舐舔。 六忌剔筷：以筷作签，入嘴剔牙。 七忌响筷：以筷敲盆，叮当作响。 八忌插筷：筷如双柱，直插碗中。

笔记本电脑给手机共享wifi热点

27 Feb 2014, 13:56
读书学习
wifi

适用范围：没有wifi，仅有连接网线上网的手提电脑。使用以下步骤配置后，手机、pad等可以使用笔记本共享的wifi上网。 打开笔记本的无线网络开关 以管理员身份打开“命令提示符” 依次输入下面2行命令 netsh wlan set hostednetwork mode=allow ssid=T410 key=1234567899 netsh wlan start hostednetwork 说明： ssid=T410 表示无线的名称为T410，key=1234567899表示无线密码为1234567899 打开网络连接能看到一个新增的无线网络连接 在本地连接上右键，然后点属性，将本地连接共享给刚才创建的 “无线网络连接 2” 完成了所有配置。每次电脑关机、开机后，重复步骤1、2，然后输入一行命令：netsh wlan start hostednetwork （说明：可以将此命令保存成bat格式的文件，需要时双击执行或者设置成开机自动执行，见下图）
